# PRIMETRADE.AI-ASSIGNMENT

This project analyzes 211,224 Hyperliquid trade executions from 32 accounts (May 2023–May 2025) against the Bitcoin Fear & Greed Index to understand how market sentiment relates to trader behavior and performance. The key finding is that sentiment barely affects win rate — which stays in a tight 76–89% band across all five sentiment regimes, with no statistically significant difference between Fear-day and Greed-day trade outcomes (Mann-Whitney p = 0.53) — but it strongly affects how traders behave: activity and position size both spike during Fear/Extreme Fear (roughly 4–6x more trades, 2x larger positions) and taper off during Greed, pointing to a contrarian "buy the dip" pattern. Long positions are the consistent profit driver regardless of sentiment (84–90% win rate in every regime), while short positions are more sentiment-sensitive, performing best specifically during fear-driven sell-offs; coin-level results further show that sentiment-based effects vary by asset, working well for BTC/ETH/SOL but breaking down for smaller, high-beta tokens. The analysis is delivered as a formatted Word report with supporting charts, a reproducible Python script, and a README tying the deliverables together.
